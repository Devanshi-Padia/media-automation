Social Media Credential Setup Guide

Instagram (via Facebook)

  1. Go to Meta for Developers
  2. Create an app → Choose “For Something Else” → Select Business
  3. In Products, add Instagram Graph API
  4. Connect your Instagram Business Account and Facebook Page
  5. Under Settings > Basic: Copy your App ID and App Secret
  6. Use Graph API Explorer to generate a Page Access Token
  7. Get your Page ID from Page Settings
Store: Instagram Username, Password, Facebook Page ID, Page Access Token

Facebook Page

Already covered in Instagram setup.

  • Page ID: Find Page ID
  • Page Access Token: via Graph API Explorer

Twitter / X

  1. Go to Twitter Developer Portal
  2. Create a Project and an App
  3. Go to Keys & Tokens section
  4. Generate:
    • API Key (Consumer Key)
    • API Secret Key
    • Access Token
    • Access Token Secret
Store: API Key, API Secret Key, Access Token, Access Token Secret

LinkedIn

  1. Go to LinkedIn Developer Portal
  2. Create a new App
  3. In Auth tab, add http://localhost as redirect URI (for local testing)
  4. Use OAuth 2.0 to generate:
    • Access Token
    • Author URN (e.g. urn:li:person:xxxx or urn:li:organization:xxxx)
  5. See: LinkedIn OAuth Flow
Store: Access Token, Author URN

Discord

  1. Open your Discord Server
  2. Go to Server Settings → Integrations → Webhooks
  3. Create a new Webhook
  4. Copy the Webhook URL
Store: Webhook URL

YouTube

  1. Go to Google Cloud Console
  2. Create a new project and enable YouTube Data API v3
  3. Navigate to APIs & Services → Credentials and create an OAuth 2.0 Client ID for Desktop or Web
  4. Note down your Client ID and Client Secret
  5. Use OAuth 2.0 Playground to:
    • Authorize https://www.googleapis.com/auth/youtube.force-ssl
    • Exchange code for access & refresh token
  6. Get Channel ID from:
    • Visit your channel page, or
    • Use API: /youtube/v3/channels?part=id&mine=true
Store: Client ID, Client Secret, Refresh Token, Channel ID

Telegram

  1. Open Telegram and search @BotFather
  2. Create a new bot using /newbot and get the Bot Token
  3. Send a message to your bot in Telegram
  4. Visit: https://api.telegram.org/bot<BOT_TOKEN>/getUpdates
  5. Extract the Chat ID from the JSON response
Store: Bot Token, Chat ID
